# thunder-market
게시판 형태의 중고거래 플랫폼을 개발한 학습용 프로젝트입니다.

# 영상
https://www.youtube.com/watch?v=xzkehthGkq4

# 기술 스택
- Back-end
  - Spring Boot 2.7.2
  - JPA
  
- Front-end
  - HTML, CSS, JavaScript
  - Thymeleaf
  - Bootstrap5
  
- Database
  - H2 database

# 기능

- 회원
  - 회원가입
  - 로그인
  - 로그아웃
  
- 게시물
  - 게시물 작성(이미지 첨부)
  - 게시물 수정
  - 게시물 삭제
  
- 댓글
  - 댓글 작성
  - 댓글 수정
  - 댓글
